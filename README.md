# AI Client Acquisition System

A powerful business engine that automates lead generation, outreach, and client acquisition for agencies and service-based businesses.

## Features

- **AI-Powered Lead Generation**: Intelligent identification and qualification of potential clients
- **Automated Outreach**: Personalized communication sequences
- **Analytics Dashboard**: Real-time insights and performance metrics
- **CRM Integration**: Seamless integration with existing CRM systems

## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm package manager

### Installation

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build & Deploy

```bash
# Build for production
pnpm build

# Start production server
pnpm start
```
